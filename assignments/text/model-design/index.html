Text branch

Model architecture

Four text classifiers: SimpleRNN, LSTM, BiLSTM, and DistilBERT, configured for 60-token (RNNs) and 128-token (Transformer) inputs.

Architecture
Models

Architectures

Models chosen to balance simplicity, recurrent memory, and pretrained context.

RNN

SimpleRNN

  • 1-layer vanilla RNN, hidden size tuned for speed.
  • Max seq length 60, vocab 33,464, trainable embeddings.
  • Baseline to show long-range limitations.
SimpleRNN: embeddings → 1-layer RNN → dropout → linear classifier.
LSTM

LSTM

  • Single-layer LSTM for longer dependencies.
  • Dropout before classifier head.
  • Best overall metrics among trained models.
LSTM: single LSTM layer with dropout before the classifier head.
Bi

BiLSTM

  • Bidirectional LSTM; concat forward/backward states.
  • Same 60-token inputs as LSTM.
  • Marginal gains in context, similar metrics to LSTM.
BiLSTM: forward/backward states concatenated for classification.
BERT

DistilBERT

  • Pretrained transformer (distilbert-base-uncased).
  • Sequence length 128; [CLS] pooled for classification.
  • Notebook uses one loader for train+val (no separate val loader).
DistilBERT: transformer encoder with pooled [CLS] for the 42-way classifier.
