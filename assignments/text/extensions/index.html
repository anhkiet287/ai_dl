Constraints

Limitations

Current bottlenecks observed in experiments.

  • Class imbalance remains; minority classes (e.g., QUEER VOICES) under-represented.
  • Transformer limited to 128 tokens; long descriptions may truncate useful context.
  • DistilBERT uses a shared train/val loader in notebook runs, risking mild leakage.
  • Minimal augmentation; robustness to sarcasm and emerging entities is untested.
Ablations

Key ablation studies

Planned checks to quantify design trade-offs.

  • Max length sweep (128 → 256 → 320) for DistilBERT/DeBERTa.
  • Loss: cross-entropy vs class-balanced vs focal.
  • Augmentation on vs off (back-translation/paraphrase) for tail labels.
  • Class-weighted sampling vs uniform in RNN loaders.
  • Learning rate and warmup schedule sensitivity for transformers.
