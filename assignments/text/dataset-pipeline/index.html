Preprocessing

Filtering, cleaning, tokenization

Normalize text then tokenize to a max length of 60 tokens.

Filtering & setup

  • Removed low-similarity / tiny categories (kept 42 classes).
  • Final vocabulary: 33,464 tokens.
  • Stopwords retained for model-aware tokenizers; ratio ~35%.
  • Max sequence length: 60 tokens (covers >99%).

Cleaning

  • Lowercasing and Unicode normalization.
  • Strip URLs, HTML entities, extra whitespace.
  • Keep punctuation that carries sentiment/topic cues.

Tokenization

  • Tokenizer: subword (e.g., WordPiece/BPE) over headline + short description.
  • Truncation: head+tail mix to preserve both fields.
  • Padding: dynamic per-batch; attention masks generated.

Training config

  • Batch size: 32
  • Shuffling each epoch; deterministic seed for splits.
  • Class imbalance handled in evaluation with Macro-F1.
Subset strategy: filtered low-similarity categories while preserving major sections.
