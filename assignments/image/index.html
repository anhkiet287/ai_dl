Image Classification

Balanced 20-class ImageNet subset (10k images: 7k/1.5k/1.5k). Seven models (3 scratch, 4 pretrained); best single model ViT-L/16-PT: Acc 1.000, Macro-F1 1.000; ViT-B/16-PT close behind at 0.998 Acc.

Task definition

Assignment Requirements

Image classification pipeline on a balanced 20-class ImageNet subset (10k images).

What we must demonstrate

  • Build an end-to-end image classification pipeline.
  • Use a valid dataset with ≥5 classes (here: 20 classes, perfectly balanced).
  • Report accuracy and macro-F1; include calibration (ECE) and robustness checks.
  • Show EDA, preprocessing/DataLoader, model design, training, evaluation, and comparisons with figures/tables.

Dataset expectations

  • Balanced splits: 7k train / 1.5k val / 1.5k test (500 per class).
  • Real images, 224×224; documented transform pipeline.
  • Train augments only; val/test use resize+centercrop+normalize.
  • Hyperparameters declared: LR, scheduler, label smoothing, MixUp/CutMix usage.
Methodology

Compared approaches

Three groups: scratch models, pretrained finetune, and ensembles.

S

Scratch models

CNN-Scratch, ResNet-18-Scratch, ViT-Scratch; 30 epochs; MixUp/CutMix per model.

P

Pretrained finetune

ResNet-50/101, ViT-B/L with 20-way heads; head-only then full finetune (5 epochs total).

E

Ensembles

2–4 model combos; top accuracy 0.9993 (no gain over best single ViT-L/16-PT).

Summary

Key findings

Main takeaways from experiments.

Pretrained ViTs dominate ViT-L/16-PT hits 1.000 test Acc / F1; ViT-B/16-PT reaches 0.998 Acc / 0.9979 F1 with far fewer params.
Scratch lag Best scratch (ResNet-Scratch) 0.7033 Acc; CNN-Scratch 0.4580; ViT-Scratch 0.1467 (data hungry).
Ensembles flat Top ensemble 0.9993 Acc; small uplift vs other pretrained but still below ViT-L.
