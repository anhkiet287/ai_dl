Task definition

Assignment Requirements

This assignment focuses on multimodal classification: given a matched image–text pair, predict the correct class label.

What we must demonstrate

Requirements for the multimodal assignment.

  • Build a multimodal classification pipeline for image + text inputs.
  • Use a valid multimodal dataset with real paired image–text samples (≥5 classes, enough size).
  • Compare zero-shot and few-shot approaches on the same task.
  • Report accuracy, and F1 when classes are imbalanced.
  • Present dataset exploration, preprocessing/DataLoader setup, model design, training/adaptation, evaluation, and comparison with tables/figures.

Dataset expectations

  • Pairs must describe the same entity/event (no random pairings).
  • At least 5 classes with sufficient samples.
  • Supports zero-shot vs. few-shot evaluation.
  • Realistic image–text distribution (not synthetic noise).
Methodology

Compared approaches

We evaluate two main paradigms: zero-shot learning (no fine-tuning) and few-shot learning (minimal labeled data).

ZS

Zero-shot classification

Use pretrained models and text prompts without any fine-tuning. Classify by semantic similarity between image embeddings and label descriptions.

FS

Few-shot learning

Few-shot prompt adaptation and CoOp on small labeled sets; compare stability, calibration, and class-wise gains.

Summary

Key findings

Main takeaways and insights from our experiments.

Best method: Prompt Adaptation

Leads overall F1 and calibration (lowest ECE/NLL/Brier); major gains on weak classes like Global Business and Television.

Persistent hard classes

Style, Media, Your Money, Well, Economy remain difficult due to semantic overlap and cross-topic cues.

Confusion families

Entertainment (Movies↔Television), lifestyle (Style↔Fashion), and finance/platform (Media/Technology/Opinion/Your Money) dominate remaining errors.

Quick reference

Method comparison summary

High-level comparison of zero-shot and few-shot approaches.

Approach Training data Fine-tuning Inference speed Main strength Main limitation
Zero-shot CLIP None (pretrained only) No Fast No labels required; interpretable saliency Lower F1; overconfident on Travel→Real Estate & Music→Theater
Few-shot Prompt Adaptation Small labeled set (few shots) Prompts + logit scale Fast Best F1 and calibration; biggest gains on weak classes Still confuses semantic neighbors (Movies/Television, Style/Fashion)
Few-shot CoOp Small labeled set (context tokens) Context prompts Medium Good gains on some classes (Global Business, Books) Brittle on Well; higher over-confidence than Prompt Adaptation

Best performers

  • 1st: Few-shot Prompt Adaptation — 0.4726 test acc
  • 2nd: Few-shot CoOp — 0.4443
  • 3rd: Few-shot CE Probe — 0.4294
  • Gap vs zero-shot: +5.2 acc pts, +6.1 macro-F1 pts

Efficiency analysis

  • Fastest training: Zero-shot (none); fastest trained: MLP probe (~0.16 h)
  • Fastest inference: Zero-shot 7.35 ms/img (Prompt Adaptation 7.49 ms/img)
  • Highest throughput: Zero-shot 136 img/s
  • Best accuracy/efficiency: Prompt Adaptation (top accuracy, near-best latency)
