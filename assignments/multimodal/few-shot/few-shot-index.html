CLIP adaptation with 10 labeled samples per class

Few-shot Multimodal Learning

We adapt a pretrained CLIP backbone using a small labeled subset of the dataset. The goal is to measure how much performance improves over the zero-shot baseline when only a few examples are available for each class.

Overview Setup Methods Results Takeaway
Overview

Few-shot setting

Same backbone, same split, small labeled budget.

  • Backbone: CLIP ViT-B/32
  • Training budget: 10 shots per class (240 training samples total)
  • Classes: 24 news categories
  • Metrics: Test Accuracy and Test Macro-F1
  • Main question: which adaptation strategy gives the best gain over zero-shot?
Setup

Shared training configuration

All methods use the same backbone and evaluation split for a fair comparison.

Common configuration

  • Frozen CLIP backbone
  • Image size: 224 × 224
  • Few-shot training on balanced per-class sampling
  • Validation-based model selection

What changes across methods

  • CE Probe: train a linear classifier
  • MLP Probe: train a nonlinear head
  • Prompt Adaptation: learn a small text-side update
  • CoOp: learn context prompts for class labels
Methods

Approaches compared

Short descriptions only, so the page stays concise.

CE Probe

Train a single linear layer on top of frozen CLIP image features. This is the simplest few-shot baseline.

MLP Probe

Add a small nonlinear head on top of frozen image features to capture slightly richer decision boundaries.

Prompt Adaptation

Keep CLIP frozen and learn a small update on the class text representation. This gives the best performance in our experiments.

CoOp

Learn context tokens for prompt embeddings while keeping the pretrained CLIP backbone frozen.

Results

Few-shot comparison

Prompt-based adaptation gives the strongest result.

Method Test Accuracy Test Macro-F1 Train Time
Zero-shot CLIP 42.05% 38.88% 0 s
Few-shot CE Probe 42.94% 40.94% 1377.18 s
Few-shot MLP Probe 42.68% 41.35% 591.48 s
Few-shot Prompt Adaptation 47.26% 44.96% 1072.22 s
Few-shot CoOp 44.43% 42.72% 1268.62 s
Takeaway

What we learned

One short conclusion block is enough here.

Few-shot learning consistently improves over the zero-shot baseline. Among the tested methods, Prompt Adaptation performs best with 47.26% accuracy and 44.96% Macro-F1. This suggests that adapting the text side of CLIP is more effective than only training a shallow classifier on frozen image features.
