Why CLIP

CLIP backbone choice

Pretrained vision–language model used as a frozen feature extractor.

What CLIP gives us

  • Strong 512-D image embeddings aligned to text space (trained on 400M image–text pairs).
  • Vision tower: ViT-B/32 style patchify (3->768), 12-layer transformer, projection 768->512.
  • Text tower: 12-layer transformer producing 512-D text embeddings for class prompts.
  • Zero-shot capability via cosine similarity of image/text features.

Why we freeze it

  • 87.8M vision params (151M total CLIP) kept frozen → fast training, low overfit risk.
  • We only train lightweight heads (=0.2M params) or prompt deltas.
  • Freezing preserves broad prior while allowing domain adaptation in heads/prompts.
  • Consistent embeddings enable swap-in of MLP, CE, few-shot similarity, or CoOp heads.

MLP head

LN → FC 512->256 → GELU + Dropout → FC 256->24.

MLP head summary
class MLPHead(nn.Sequential):
    def __init__(self):
        super().__init__(
            nn.LayerNorm(512),
            nn.Linear(512, 256),
            nn.GELU(),
            nn.Dropout(0.2),
            nn.Linear(256, 24),
        )
  • Params: 138,520 · Trainable.
  • Two-layer projection with GELU + dropout.
  • Optimized on frozen CLIP embeddings.
  • Use for supervised baselines.

CE linear head

Single projection 512->24 for cross-entropy training.

CE head summary
class CEHead(nn.Linear):

CoOp head

Learnable context tokens → CLIP text encoder → similarity scoring.

CoOp classifier (trainable prompts, frozen CLIP)
# forward (simplified)
image = encode_image(x) # frozen CLIP vision
prompt_embeds, tok = prompt_learner() # trainable ctx
text = text_encoder(prompt_embeds, tok) # frozen CLIP text
scale = logit_scale.exp().clamp(max=100)
logits = scale * image @ text.T
  • PromptLearner: shared n_ctx tokens (trainable) + fixed token embeddings.
  • TextEncoderHF: 12-layer CLIP text transformer (self-attn + MLP 512->2048->512), final LN, text_projection 512->512 (frozen).
  • Output text features (24,512), L2-normalized; optional trainable logit_scale.
  • Text encoder params (frozen): 63,428,096.
  • Only prompt ctx (and optional scale) are trained.

Prompt similarity

Normalized image features vs (base + delta) text features with learned logit scale.

Few-shot prompt similarity head (12,289 params)
image = F.normalize(encode_image(x), dim=-1)
text = F.normalize(base_text + text_delta, dim=-1)
scale = logit_scale.exp().clamp(max=100)
logits = scale * image @ text.T
  • Head params: 12,289 (text_delta + optional logit_scale).
  • Logits = exp(logit_scale) · L2(image_feat) · L2(text_feat)^T.
  • Backbone remains frozen; only text_delta / scale train.

Takeaway

We keep CLIP frozen as a feature extractor and swap lightweight heads: supervised MLP/CE probes or prompt-based similarity heads (few-shot, CoOp). This isolates downstream learning to =0.2M params while leveraging 150M frozen CLIP parameters.
