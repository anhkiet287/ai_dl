# Deep Learning Course GitHub Pages Template

This template is ready to upload to a GitHub repository and publish with GitHub Pages.

## Included files
- `index.html`: common homepage for the whole course
- `assets/styles.css`: shared styling
- `assignments/assignment1.html`: detailed page for Assignment 1
- `assignments/assignment2.html`: placeholder page
- `assignments/assignment3.html`: placeholder page

## What to edit
1. Group name
2. Member names
3. GitHub repository links
4. Demo / presentation video links
5. Google Drive or report links
6. Result tables and summaries on Assignment 1 page

## Publish with GitHub Pages
1. Create a public GitHub repository
2. Upload all files from this folder to the repository root
3. Go to Settings -> Pages
4. Choose Deploy from a branch
5. Select `main` and `/ (root)`
6. Save and wait for the site URL
